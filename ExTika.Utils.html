    trim(s)

    Specs

    trim(String.t) :: String.t

    Returns a string where leading/trailing Unicode whitespace has been removed. Works on versions of Elixir before 1.3 which don’t have the String.trim function.

    Examples

    iex> ExTika.Utils.strip("   abc   ")
"abc"
